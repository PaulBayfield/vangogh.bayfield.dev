# 🚀 • Presentation

This project is a personal website that showcases the life and works of Vincent van Gogh. It was created as a way to document and share the experience of visiting the Musée d'Orsay exhibition on Van Gogh's final months in Auvers-sur-Oise. The website features a timeline of Van Gogh's major works, along with information about his life and artistic journey.
